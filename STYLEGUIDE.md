## **Types of Variables**
- Final variables
- Public variables
- Private variables
- Static variables
- other variables



  
### Final Variables

Final variables are used to show and ensure that this declared variable will never change within the programming process

Final variables format prefered in this team:
`final [datatype] [VARIABLENAME] = [assigned value];`


**Ensure that Variable Name of a final variable is all caps**



  
### Static Variables

Static variables are used to allow other classes to successfully access declared variable

Static variables format prefered in this team:
`static [datatype] [VARIABLENAME] = [assigned value];`

**Static variables should be all caps, and could be combined with public and private variables**

`public static [datatype] [VARIABLENAME] = [assigned value];`



  
### Any Other Variables

1. Public variables are used to grant other classes access to the declared variable
2. Private variables are used to ensure that no other classes can directly access the declared variable
3. normal variable, simply declaration of a variable with no special functions.

Public variables format prefered in this team:
`public [datatype] [variableName] = [assigned value];`

Private variables format prefered in this team:
`private [dataType] [variableName] = [assigned value]`

Normal variables fomat prefered in this team;
`[datatype] [ variableName] = [assigned value]`

**Ensure that the first letter of the variable is lowerCase. Everytime a new word comes, the first letter of a new word shall be capitalized.**
